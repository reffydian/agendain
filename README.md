# Agendain - Aplikasi Manajemen Jadwal 📱
Aplikasi untuk manajemen jadwal harian kalian.

## Fitur ✨
- Login
- Manajemen Jadwal:
    - Tambah jadwal
    - Lihat daftar jadwal
    - Detail jadwal
- Navigasi:
    - Bottom navigation bar
    - Floating Action Button

## Instalasi ⚙️
```bash
git clone https://github.com/reffydian/agendain.git
cd agendain
flutter pub get
flutter run
```
